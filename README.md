# React - Vite - TailwindCSS Starter Template

This is a Starter Template with Vite, React and Tailwind CSS I use for a lot of my own projects. It is packaged with various tools and libraries that I use in just about every projects!

![preview](image.png)

## Packages included :

- [Daisy Ui Tailwind (TailwindCSS Component Library)](https://daisyui.com/docs/)
- [React Router Dom](https://reactrouter.com/en/main)
- [React Icons](https://react-icons.github.io/react-icons/)
- [Framer Motion](https://www.framer.com/motion/)
- [Axios](https://axios-http.com/docs/intro)
